# ae-chat-room

> a chat room demo, built with React and Socket.io.

## Installation

```sh
npm install
```

## Development or Debug

- server (endpoint: `http://127.0.0.1:3030/sockets`)

```sh
npm run server
```

- client `http://127.0.0.1:3000`

```sh
npm run client
```

## Deploy

server & client `http://127.0.0.1:4000`

```sh
npm run deploy
```
